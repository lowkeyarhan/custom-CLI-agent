# lowkeyarhan

A CLI coding agent that helps you write code, fix bugs, and explore your codebase directly from the terminal.

## Installation

```bash
npm install -g lowkeyarhan
```

## Usage

```bash
lowkeyarhan "Create a simple HTTP server in Node.js"
```

## Configuration

Create a `.env` file or export the following environment variable:

```bash
export OPENROUTER_API_KEY=your_key_here
```

## Features

- **Autonomous**: Can read files, write code, and execute commands to achieve the goal.
- **Tools**: Built-in tools for file manipulation and command execution.
- **History**: Maintains conversation history for context awareness.

## Development

```bash
npm install
npm run build
npm start -- "Your task here"
```
